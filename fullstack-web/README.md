# Fullstack-web

| ------------------- | ----------------------------- |
| React               | Framework                     |
| Formik              | React Forms                   |
| Urql                | Graphql client                |

`npm i -D @graphql-codegen/typescript @graphql-codegen/cli @graphql-codegen/typescript-urql`
